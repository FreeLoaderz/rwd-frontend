Who is Feeloaderz?

Freeloaderz is a coalition of Cardano stake pools offering Cardano Submit API access to their relay servers to help improve transaction speed by opening up ports to user wallets. Join the movement in our Discord Server.

What is SmartClaimz?

The current cost of processing transactions for airdrop and token distribution is high and often takes a long time. This can result in a frustrating and distressing user experience. Often times, it isn't clear if the transaction was even submitted correctly. SmartClaimz addresses these user experience issues with a faster/clearer transaction with less fees overall. This is achieved by:
  • Freeloaderz SPOs providing the Cardano Submit API end point on their pool Relays
  • A load balancer providing a round robin distribution to these Relays for transactions
  • Native script processing to allow for parallel transactions
  • The ability to accumulate your rewards instead of having to claim every epoch (less network transactions overall)
  • Web3 dApp wallet connection
  • Optimized UTXO management

How does SmartClaimz work?

SmartClaimz is powered by ©Drasil-Core. SmartClaimz uses the Cardano "Reward & Distribution Application" created and run by ©Drasil. This application is part of an application suite designed to power-up developers. With Drasil, Cardano dApps can be developed easier, faster, and more secure. The Reward & Distribution Application functionality includes one click instant claims, claim history, automated reward calculations and distribution, and staking delegation. This is all neatly integrated with the SmartClaimz web app. The SmartClaimz website is an open source Angular project available on the Freeloaderz Github. If you have any concerns regarding how your wallet is used, you can view the code and see exactly how we are using it.
