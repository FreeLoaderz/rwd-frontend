Testnet Tools and Information

To participate in SmartClaimz for Testnet, delegate your Testnet wallet to one of the following Testnet pools.

Apex Pool (APEX)

Panl Stake Pool (PANL)

Lido Nation (LIDO)

At any time, you can immediately generate random test rewards for your wallet by using this "Generate Rewards" button.

**Rewards for each epoch will eventually be automatically calculated. If you don't see your rewards, it may be that the reward script hasn't been run yet.
